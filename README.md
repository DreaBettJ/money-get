# money-get

A股投资辅助 Agent - 历史操作复盘 + 数据分析

## 核心功能

| 功能 | 说明 |
|-----|------|
| 历史操作复盘 | 记录买卖，分析胜率、盈亏比 |
| 历史数据LLM分析 | 用大模型分析K线、成交量、板块轮动 |
| 舆情+政策监控 | 利好/利空消息，政策解读（待开发） |
| 财报解读 | 自动抓取财报，提取关键指标（待开发） |
| 上下文隔离 | 按主题/股票分割记忆，防止爆炸 |

## 技术栈

- Python
- LangChain / LangGraph
- MiniMax API

## 架构

```
数据层 → Agent层 → 输出层
```
